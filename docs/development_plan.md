# Development Plan

## 1. 项目目标


项目名称：

Power-Agent

基于大语言模型 Agent 的电厂智能预警与故障诊断系统。


开发周期：

8周


最终目标：

完成一个可运行 Demo，实现：

- 电厂运行数据模拟
- 异常检测
- 故障诊断
- Agent智能问答
- 可视化展示
