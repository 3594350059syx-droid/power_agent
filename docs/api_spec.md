
# API Specification

## 1. 基本规则

系统模块之间通过 HTTP API 通信。

统一规范：

- 协议：HTTP
- 数据格式：JSON
- URL格式：

/api/v1/{模块}/{功能}


示例：

/api/v1/agent/diagnose


---

## 2. 请求规范

请求格式：

{
    "参数名": "参数值"
}


示例：

{
    "device_id": "A001",
    "time": "2026-01-01"
}


---

## 3. 返回规范

统一返回：

{
    "success": true,
    "message": "说明信息",
    "data": {}
}
